## Quick Start

1. Clone the repository:

```bash
git clone https://github.com/willy264/HealthCare_dash.git
```

2. Install dependencies:

```bash
npm install
# or
pnpm install
```

3. Start the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) to view the dashboard.
